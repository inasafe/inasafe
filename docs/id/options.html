Pilihan¶

InaSAFE plugin menyediakan dialog pilihan yang memungkinkan Anda untuk menentukan berbagai pilihan yang berhubungan untuk bagaimana InaSAFE akan berperilaku. Dialog pilihan dapat ditampilkan dengan mengklik pada InaSAFE simbol plugin toolbar pilihan (seperti yang terlihat di bawah) atau dengan melakukan Plugins ‣ InaSAFE ‣ InaSAFE Options.

Pilihan ikon

Kemudian dialog ini akan muncul:

Seperti yang dapat anda lihat, dialog ini terbagi menjadi tiga tab:

  • Pilihan Dasar
  • Pilihan Template
  • Pilihan Lanjutan

Dialog Pilihan

Berikut ini pilihan yang tersedia di dalam Options Dialog:

Pilihan Dasar¶

  • Hanya tampilkan layer yang terlihat di InaSAFE : Pilihan ini akan menentukan apakah seluruh layer ancaman dan layer dampak harus terdaftar pada kotak pilihan InaSAFE (ketika dicentang) atau hanya layer yang terlihat.

  • Gunakan ‘judul’ pada Editor Kata kunci sebagai nama layer QGIS : Pilihan ini akan menyebabkan QGIS untuk memberi nama layer pada Pohon layer menggunakan kata kunci judul pada berkas kata kunci layer tersebut. Jika layer tidak memiliki ‘judul’ pada kata kuncinya, atau tidak memiliki kata kunci sama sekali, maka penamaan layer pada QGIS akan berlaku secara normal.

  • Zoom ke lapisan dampak saat perhitungan dampak selesai : Pilihan ini akan menyebabkan tampilan peta untuk diperbesar/diperkecil sehingga benar-benar berisi InaSAFE hasil skenario peta dampak ketika analisis selesai.

  • Sembunyikan layer keterpaparan saat perhitungan dampak selesai : Pilihan ini akan menyebabkan QGIS untuk mematikan layer keterpaparan yang sedang digunakan ketika InaSAFE melengkapi analisis ini. Anda dapat mengaktifkan kembali visibilitas layer lagi dengan mencentang kotak centang di legenda.

  • Saat melakukan analisis, dataset yang digunakan hanya sebatas yang tampil di layar : Pilihan ini akan menyebabkan QGIS memotong layer ancaman dan keterpaparan yang terlihat pada peta kanvas.

  • Potong fitur saat melakukan clipping (akan memotong poligon menjadi lebih kecil): Pilihan ini, setiap poligon yang melampaui luasan analisis akan dipotong sehingga mereka berisi oleh luasan analisis. contohnya jika Anda memiliki layer poligon ancaman banjir yang meluas jauh melampaui area yang Anda minati, poligon banjir akan berkurang sehingga hanya bagian di dalam area yang Anda minati dipertahankan. Ini dapat mempercepat waktu pemrosesan.

  • Tampilkan layer intermediate yang dibuat pascapemrosesan : Pilihan ini akan menyebabkan QGIS menampilkan file menengah yang dihasilkan oleh langkah-langkah postprocessing pada peta kanvas.

  • Nilai default perbandingan wanita : Ini adalah pengaturan bawaan nilai perbandingan wanita terhadap total populasi yang digunakan untuk menghasilkan rincian berdasarkan jenis kelamin.

Pilihan Template¶

  • Logo organisasi: Pilihan ini digunakan untuk menentukan logo yang akan digunakan sebagai logo organisasi pada berkas pdf proyek skenario dampak anda. Lihat Laporan pada Panduan Pengguna InaSAFE untik informasi lebih lanjut mengenai laporan.

  • Tampilkan logo organisasi pada panel utama: Jika pilihan ini diaktifkan maka logo organisasi yang dipilih (jika ada) akan ditampilkan di bagian bawah panel utama inasafe.

  • Direktori template laporan: Pilihan ini digunakan untuk menentukan sebuah direktori dengan template rancangan anda sendiri. Template dari direktori ini akan tersedia untuk digunakan pada saat menghasilkan laporan untuk proyek skenario dampak anda. Lihat Laporan pada Panduan Pengguna InaSAFE untuk informasi lebih lanjut mengenai laporan.

  • Teks disclaimer organisasi: Pilihan ini digunakan untuk merubah pengaturan bawaan teks penolakan yang akan dicetak dalam laporan. Cukup masukan teks anda sendiri dan teks tersebut akan digunakan. Lihat Laporan pada Panduan Pengguna InaSAFE untuk informasi lebih lanjut mengenai laporan.

Pilihan Lanjutan¶

  • Cache kata kunci (keyword) untuk sumber data jarak jauh : Opsi ini digunakan untuk menentukan di mana kata kunci akan disimpan untuk dataset apabila tidak dimungkinakn untuk ditulis ke dalam file .keywords. Lihat Sistem Kata kunci pada Panduan Pengguna InaSAFE untuk informasi mengenai sistem kata kunci.

  • Bantu pengembangan InaSAFE dengan mengirimkan kesalahan kepada remote server : Pilihan ini, ketika diaktifkan, akan mengirimkan informasi diagnosis kembali ke server proyek InaSAFE di setiap kesalahan kita mampu menjebaknya. Informasi yang diberikan berguna bagi tim InaSAFE untuk meningkatkan ketahanan perangkat lunak yang kami produksi dan untuk melihat tren pada isu-isu yang dihadapi orang. Silakan catat teks peringatan dibawah pilihan ini yang disertakan disini secara penuh:

Catatan

Untuk mengaktifkan/menonaktifkan pengaturan di atas membutuhkan QGIS untuk di restart . Pesan kesalahan dan informasi diagnostik akan terkirim ke http://sentry.linfiniti.com/inasafe-desktop/ . Beberapa institusi mungkin tidak memperbolehkan anda untuk mengaktifkan fitur ini - periksa kembali dengan administrasi jaringan anda jika tidak yakin. Meskipun data yang terkirim adalah anonim, informasi yang terkandung dalam tracebacks mungkin memiliki jejak sistem berkas dimana akan menunjukan identitas anda atau informasi lain dari sistem anda.

  • guilabel:Aktifkan mode pengembang : Ketika pilihan ini diaktifkan, maka anda dapat melakukan klik kanan pada area laporan dan menggunakan alat webkit debug untuk melakukan debug pada hasil. Pilihan ini dimaksudkan hanya untuk pengguna yang mahir.

  • Gunakan zonal statistik QGIS : Dengan tombol ini, anda dapat merubah antara menggunakan zonal statistik QGIS internal atau zonal statistik yang tersedia dalam InaSAFE

Catatan

Anda dapat klik tombol Help beberapa waktu dan ini akan membuka bantuan pencarian dokumentasi untuk halaman ini

Catatan

Dengan menekan Batal maka dialog pilihan akan menutup dan perubahan apa saja tidak akan diterapkan. Tekan OK maka kotak dialog Pilihan akan menutup dan perubahan apa saja akan segera diterapkan.

Catatan

Urutan tombol yang tepat yang ditampilkan pada kotak dialog ini mungkin berbeda tergantung pada sistem operasi atau lingkungan desktop Anda.
