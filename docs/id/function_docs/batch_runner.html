The Batch Runner lets you load saved scenarios (Save Scenario) and batch run them in one go:

Batch Runner

Before running the Batch Runner you might want to use the Save current scenario tool to first save some scenarios on which you can let the batch runner do its work. This tool lets you run saved scenarios in one go. It lets you select scenarios or let run all scenarios in one go.

Contact info@inasafe.org for more information. Source code available on GitHub! Please file a ticket if you have discovered an issue.
