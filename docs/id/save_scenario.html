Alat penyimpan skenario ini memungkinkan Anda menyimpan skenario (tb_save_scenario) dan melakukan batch run lagi dalam alat batch runner (Batch Runner) dalam satu kali jalan:

Simpan skenario saat ini

Ini adalah alat yang Anda butuhkan untuk menyiapkan/menyimpan skenario untuk alat tb_batch_runner. Ini memungkinkan Anda menyimpan skenario yang terlihat saat ini di QGIS ke dalam berkas .txt. Setelah Anda simpan, anda dapat membuka kembali file tersebut dalam Batch runner dan menghitung ulang kembali.

Deskripsi detail yang lebih jelas mengenai batch runner tersedia di Batch Runner

Catatan

Sekarang alat ini tidak menyimpan parameter fungsi dampak yang telah anda tetapkan. Oleh karena itu semua fungsi dampak akan dijalankan menggunakan pengaturan bawaan.
