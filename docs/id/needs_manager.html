Pengaturan Kebutuhan Minimum¶

Dalam versi 2.2, sistem standar kebutuhan minimum telah mengalami banyak perbaikan. Pengaturan kebutuhan minimum adalah sebuah piranti (tool) baru yang memperkenankan Anda untuk menentukan ‘profil’ dari kebutuhan minimum untuk membuatnya sesuai dengan yang diterapkan di negara atau daerah Anda.

Pengaturan Kebutuhan Minimum

Kami memuat InaSAFE dengan profil standar untuk:

  • Indonesia - berdasarkan pada ‘Perka 7/2008’
  • Filipina (berdasarkan pada ‘Panduan pada Pusat Koordinasi Evakuasi dan Surat Edaran Manajemen Bersama No. Series of 2013’)

Terdapat dua konsep utama yang harus diperhatikan dalam mengatur kebutuhan minimum:

  • profil
  • sumber

Profil Kebutuhan Minimum¶

Sebuah profil merupakan kumpulan dari berbagai sumber yang menjelaskan tentang kebutuhan minimum untuk sebuah negara atau daerah. Secara khusus sebuah profil harus berdasarkan pada sebuah standar regional, nasional atau internasional. Definisi sebenarnya dari sumber mana yang dibutuhkan dalam pembuatan profil tergantung dari kondisi wilayah dan area yang ada dimana rencana kontinjensi dirancang.

Contohnya di timur tengah, nasi merupakan makanan pokok dimana di Afrika Selatan, jagung merupakan makanan pokok dan maka dari itu rencana kontinjensi harus sesuai dengan perbedaan kebutuhan-kebutuhan semacam ini.

Sumber Kebutuhan Minimum¶

Setiap hal di dalam profil kebutuhan minimum merupakan sumber. Setiap sumber dideskripsikan sebagai sebuah kalimat bahasa alami yang sederhana seperti:

“Setiap individu harus disediakan 2,8 kg beras setiap minggunya.”

Dengan meng-klik pada isian sumber di dalam jendela profil, dan kemudian klik pada ikon pensil hitam Anda akan dapat mengatur sumber referensi perhitungan analisis menggunakan sumber referensi yang dipilih oleh pengatur (editor). Alternatif lainnya, Anda dapat membuat sumber baru untuk sebuah profil dengan meng-klik pada tanda + hitam yang terdapat pada pengaturan profil. Anda juga dapat menghilangkan sumber dari sebuah profil menggunakan tanda - pada pengaturan profil.

Pengatur Sumber

Ketika pindah ke mode ubah atau tambah sumber, pengaturan kebutuhan minimum akan berubah menjadi seperti potongan tampilan di atas. Setiap sumber akan dideskripsikan dalam beberapa istilah seperti:

  • nama sumber misalnya Nasi
  • sebuah penjelasan mengenai sumber misalnya makanan pokok
  • satuan dimana sumber diberikan misalnya kilogram
  • bentuk majemuk dari satuan misalnya kilogram
  • singkatan untuk satuan misalnya kg
  • alokasi default untuk sumber misalnya 2.8. Anka ini dapat diganti setiap kali analisis.
  • minimal diperbolahkan yang digunakan untuk mencegah pengalokasian seperti tidak ada air minum untuk pengungsi
  • maksimal diperbolehkan yang digunakan untuk mengatur batas atas yang masuk akal untuk sumber
  • sebuah kalimat terbaca yang digunakan untuk mengkompilasi kalimat untuk menjelaskan sumber di laporan.

Parameter ini mungkin semuanya sudah cukup jelas, tapi kalimat terbaca mungkin membutuhkan penjelasan lebih detil. Kalimat dikompilasi menggunakan sistem penggantian token keyword sederhana. Token-token berikut dapat digunakan:

  • {{ Default }}
  • {{ Unit }}
  • {{ Units }}
  • {{ Unit abbreviation }}
  • {{ Resource name }}
  • {{ Frequency }}
  • {{ Minimum allowed }}
  • {{ Maximum allowed }}

Ketika token diletakan di kalimat, token akan digantikan dengan nilai aslinya saat pembuatan laporan. Contoh berikut menunjukan kalimat bertoken yang memuat semua pilihan keyword:

Pengungsi seharusnya disediani dengan {{ Default }}
{{ Unit }}/{{ Units }}/{{ Unit abbreviation }} dari {{ Resource name }}.
Meskipun tidak kurang dari {{ Minimum allowed }} dan tidak lebih dari
{{ Maximum allowed }}. Ini seharusnya diberikan {{ Frequency }}.

Akan menghasilkan kalimat yang mudah dibaca seperti ini:

Pengungsi seharusnya disediani dengan 2.8
kilogram/kilogram/kg dari beras.
Meskipun tidak kurang dari  0 dan tidak lebih dari
100. Ini seharusnya diberikan perhari.

Setelah Anda mengisi bagian-bagian dari sumber, klik tombolSimpan sumber untuk kembali ke tampilan profil. Anda akan melihat bahwa sumber baru telah ditambahkan di daftar profil sumber.

Mengelola profil¶

Sebagai tambahan profil-profil yang standar di InaSAFE, Anda dapat membuat yang baru, bisa dari awal, atau berdasarkan yang sudah ada (yang dapat anda modifikasi).

Gunakan tombol Baru untuk membuat profil baru. Ketika diminta, berikan nama profil misalnya :keyboard:`JakartaProfile`.

Catatan: Profil harus disimpan di folder home Anda di .qgis2/minimum_needs supaya InaSAFE bisa mendeteksinya.

Cara lain untuk membuat profil baru adalah dengan menggunakan Simpan Sebagai untuk mengopi profil yang sudah ada. Profil hasi pengkopian dapat diubah sesuai kebutuhan spesifik Anda.

Profil Aktif¶

Ini sangat penting untuk dicatat, bahwa profil manapun yang Anda pilih dari daftar Profile, akan digunakan sebagai aktif dan akan digunakan sebagai basis untuk semua kebutuhan minimum analisis..
