OpenStreetMap Downloader¶

Tool ini akan membantu Anda memperoleh data (‘struktur’) bangunan dan jaringan jalan (‘jalan’) dari projek OpenStreetMap.

OpenStreetMap Downloader

Tool ini membutuhkan koneksi internet karena tool ini mengambil data dari layanan situs Laporan OSM Linfiniti

Data, setelah didownload akan tersedia untuk Anda sebagai sebuah shapefile. Sebuah gaya file secara otomatis dibuat sehingga dilambangkan dengan baik dalam QGIS. Selain itu, metadata kata kunci yang benar dibuat untuk dataset didownload sehingga dapat digunakan secara langsung dalam skenario analisis dampak menggunakan InaSAFE.

Catatan

Layanan download ini memiliki keterbatasan terhadap besarnya dataset yang dapat diterima. Kemungkinan Anda akan mendapati isu ketika mencoba untuk mengambil misalnya untuk data batas tapak bangunan seluas satu negara. Umumnya pen-download-an dataset pada level perkotaan dan di bawahnya seharusnya mampu bekerja lebih baik.

Untuk menggunakan alat ini secara efektif:

  • Gunakan QGIS untuk memperbesar ke daerah yang Anda ingin dapatkan datanya. Untuk mendapatkan area yang Anda inginkan, anda bisa mengatur kotak secara manual di area yang disediakan pada bagian bawah dari dialog.

  • Periksa apakah direktori output sudah benar. Catatan bahwa set data yang tersimpan akan diberi nama buildings.shp dan roads.shp (beserta berkas-berkas yang terhubung dengannya)

  • Secara default, nama file yang sederhana akan digunakan (misalnya roads.shp, buildings.shp). Anda dapat menggunakan prefix untuk ditambahkan di depan nama default ini,. Sebagai contoh, menggunakan prefix ‘padang-‘ akan mengubah file yang di-download menjadi ‘padang-roads.shp’ dan ‘padang-buildings.shp’. Sebagai catatan, prefix hanya dibolehkan untuk menggunakan karakter A-Z, a-z, 0-9 dan karakter ‘-‘, dan ‘_’. Anda dapat mengosongkan pilihan ini.

  • Tekan OK dan tunggu data untuk diambil.

Peringatan

Jika dataset sudah ada di direktori output, dataset tersebut akan ditimpa. Alat ini memerlukan koneksi internet untuk bekerja dan mengambil bangunan akan menggunakan bandwidth Anda.

Catatan

Data yang didownload adalah hak cipta kontributor OpenStreetMap (klik untuk info lebih lanjut).

Data bangunan¶

InaSAFE mendukung penggunaan data bangunan pada berbagai macam fungsi dampak. Pada umumnya kami mendukung batas tapak bangunan yang bersumber dari proyek OpenStreetMap (OSM)

Tipe bangunan berikut ini yang berasal (tergantung dari berbagai tags yang diberikan pada OSM) adalah sebagai berikut:

  • Sekolah

  • Universitas/Kampus

  • Pemerintahan

  • Klinik/Dokter

  • Rumah Sakit

  • Stasiun Pemadam Kebakaran

  • Kantor Polisi

  • Bangunan Publik

  • Tempat Ibadah - Islam

  • Tempat Ibadah - Unitarian

  • Tempat Ibadah - Buddha

  • Tempat Ibadah - Unitarian

  • Supermarket

  • Permukiman

  • Fasilitas Olahraga

  • Tempat Beribadah

  • Komersil

  • Industri

Jenis turunan akan hadir dalam sebuah field yang disebut ‘type’ saat men-download data bangunan. Perhatikan bahwa hal ini tergantung pada daerah mana yang anda download, kemungkinan hanya beberapa dari kategori ini yang akan muncul pada set data bangunan Anda.

Data jalan¶

InaSAFE mendukung penggunaan data bangunan untuk berbagai fungsi dampak. Pada umumnya kami mendukung jalan yang bersumber dari proyek OpenStreetMap (OSM)

Tipe jalan berikut ini yang berasal (tergantung dari berbagai tags yang diberikan pada OSM) adalah sebagai berikut:

  • Jalan besar / jalan bebas hambatan (termasuk ‘trunk’ / jalan penting dari OSM)

  • Percabangan jalan besar

  • Jalan primer

  • Percabangan jalan primer

  • Jalan tertier

  • Percabangan jalan tertier

  • Jalan Sekunder

  • Percabangan jalan sekunder

  • Jalan, perumahan, permukiman, etc. (termasuk semua jalan yang diberi tag ‘living_street’, ‘residential’, ‘yes’, ‘road’, ‘unclassified’, ‘service’, ‘’ dan NULL)

  • Track

  • Jalur sepeda, jalan kecil, etc. (termasuk jalan dengan tag ‘cycleway’, ‘footpath’, ‘pedestrian’, ‘footway’ dan ‘path’ )

Catatan

Data jalan telah disimbolisasi menggunakan aturan-aturan yang disesuaikan dengan yang ada di QGIS. Seperti bila Anda mencoba untuk memperkecilnya hingga terlihat jalan-jalannya akan disesuaikan dengan gaya (style) yang berbeda untuk menghindari peta terlalu nampak berantakan dalam skala yang lebih kecil.
