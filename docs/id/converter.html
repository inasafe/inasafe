Pengonversi ShakemapÂ¶

Peta guncangan berguna untuk melaksanakan perencanaan kontinjensi dalam kejadian gempa bumi. Biasanya hal ini didistribusikan sebagai file grid.xml yang tidak dapat digunakan dalam InaSAFE atau QGIS. Alat ini akan mengimpor file grid.xml sebagai file GEOTIFF yang nantinya dapat digunakan dalam InaSAFE.

Dua algoritma interpolasi yang berbeda dapat digunakan selama proses impor - Nearest Neighbour dan Inverse Distance. Setelah konversi, alat tersebut secara otomatis membuat metadata katakunci InaSAFE untuk layer sehingga dapat segera digunakan untuk analisis.

Konverter

Untuk menggunakan alat ini secara efektif:

  • Pilih grid.xml untk layer input.

  • Tambahkan judul shakemap yang akan digunakan untuk kata kunci.

  • Tambahkan sumber shakemap yang juga akan digunakan untuk kata kunci.

  • Pilih lokasi untuk menulis layer output.

  • Pilih algoritma interpolasi yang harus digunakan ketika mengubah grid xml menjadi raster. Jika tidak yakin simpan sebagai default.

Anda dapat memperoleh data guncangan secara gratis dari situs peta guncangan USGS.

Jika Anda mendownload file grid. Anda harus klik-kanan file ini dan pilih Simpan sebagai untuk memastikan bahwa file ini disimpan dengan benar. Pengguna OSX dan Linux juga dapat mempertimbangkan untuk menggunakan perintah tool garis untuk mengambil file grid:

OSX:

curl -O http://earthquake.usgs.gov/earthquakes/shakemap/global/shake/<shake id>/download/grid.xml

Linux:

wget -C http://earthquake.usgs.gov/earthquakes/shakemap/global/shake/<shake id>/download/grid.xml

Catatan

Ganti <shake id> dalam perintah di atas dengan identitas guncangan yang sesungguhnya.
