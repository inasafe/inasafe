Batch Runner

Dengan alat ini Anda dapat mengatur berbagai skenario dan menjalankan mereka semua dalam satu kesempatan. Sebuah kasus penggunaan yang khas mungkin ketika Anda menetapkan sejumlah skenario misalnya skenario dampak banjir semua skenario menggunakan dataset standar misalnya flood.shp. Sebagai data banjir baru telah tersedia Anda mengganti flood.shp dan kembali menjalankan skenario menggunakan batch runner. Menggunakan pendekatan ini Anda dapat dengan cepat menghasilkan rencana kontingensi daerah dengan pemahaman Anda akan perubahan-perubahan bencana. Ketika Anda menjalankan skenario batch, laporan pdf akan dihasilkan otomatis dan semua ditempatkan dalam sebuah direktori umum sehingga memudahkan Anda untuk mencari dan menyebarkan laporan tersebut.

Ketika proses batch telah selesai, proses tersebut juga akan menghasilkan ringkasan laporan seperti ini:

InaSAFE Batch Report File
-----------------------------
P: gempa bumi Sumatran fault (Mw7.8)
P: gempa di Yogya tahun 2006
P: banjir jakarta 2007
P: Tsunami di Maumere (Mw 8.1)
P: gempa Mw6.5 Palu-Koro Fault
P: gunung merapi meletus
-----------------------------
Total passed: 6
Total failed: 0
Total tasks: 6
-----------------------------

Untuk pengguna tingkat lanjut, tersedia juga pilihan untuk menjalankan script python menggunakan alat ini, tapi ini masih dianggap sebagai fitur ** eksperimental ** pada tahap ini.

Sebelum menjalankan Batch Runner anda mungkin ini menggunakan alat tb_save_scenario untuk menyimpan beberapa skenario yang ingin anda jalankan di batch runner. Alat ini memungkinkan anda untuk menjalankan semua skenario yang telah disimpan sekaligus. Ini juga memungkinkan anda memilih skenario atau menjalankan semua skenario sekaligus.
